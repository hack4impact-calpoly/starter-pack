Connect with Links

So far, we have a few individual HTML webpages right now, but many unconnected webpages doth not make a website. Like everything on the Internet, it's time we connect our webpages together with links. That's what the HyperText in HTML is referring to: a system of links.

We can make a link with an anchor tag, <a>, which will let us determine the link text as well as the destination for the link. The destination is set using a URL in the href="" attribute of the anchor tag.

Let's add a link to your index.html page that points back to your index.html page. Silly, but necessary. In the <h1> heading in your <nav> tag, wrap the text in an anchor tag.

Then, in your anchor tag's opening tag, add the href="index.html"attribute.

Lists

      
There are two types of lists we will focus on. Ordered lists and unordered lists. Both lists hold list
        items, but ordered lists precede each item with a incrementing numbers while unordered lists usually use bullet
        points.

      
Both lists have the same structure with <ol> tags representing an
        ordered list and <ul> tags representing and unordered list. Inside each
        of the list tags are list item tags, <li>, surrounding content.

Images

      
All this text we have doesn't really showcase the multimedia nature of the internet. Let's add an image to our
        index.html so that the world can percieve us! In your index.html, add the following to
        your <div class="about-image"> tag.
      

      
<img src="<path/to/image>" alt="<description of image>">
      

      
Note: Image is a self-closing tag like <br> so you just need an one tag.

      
Make sure to put an image in the same directory as all your HTML and other files! We have an image
        sky.jpg which we will add to our page.
      

      
<img src="sky.jpg" alt="A video of an image of the sky and clouds being too
          big">

      
      
Our image is way too big, but we can change it to fit better once we learn CSS. For now let's put a
        width and/or height on our image using the width and height HTML attributes to keep
        the image smaller.
      

      
Note: Always add an alt attribute value to increase accessibility
        for
        users of your website.
