Lab 2 Requirements

Goals

  • Understand the requirements for the Starter Pack

Laptop

Before you get started you will need a laptop or computer! A laptop will be used write code, test software, and access online resources. You'll want to make sure your system meets the minimum requirements below.

Operating System
  • Windows 10 and 11 (32-bit and 64-bit)
  • macOS with Apple security update support. This is typically the latest release and the two previous versions.
RAM
  • 4gb or more

Note: If you have trouble accessing a computer, you can reserve a laptop from Cal Poly Tech Rentals!
