HTML

We'll come back to Git in a little bit, but for now let's learn about the HyperText Markup Language. HTML is one half of what makes up a static website. If CSS is the pretty neon lights and JavaScript is the flashing patterns and programmed functionality, HTML is all the wiring and framework underneath. HTML is the foundation and structure for all web applications.

It may sound intimidating, but HTML is just text with some fancy formatting features. In fact, all you need to start writing HTML is a text editor.

You could write it in Word or Google Docs or Notepad if you really want the raw experience. We'll be using Visual Studio Code for comfort and ease of use.

View Your HTML

To see your spiffy new website, open a browser and type in the file path of your file. For example: /Users/hack4impact/hack4impact-calpoly.github.io/index.html on macOs.

You can also go into your File Explorer or Finder and open the file from there. Or, go to the menu bar of your browser and under File > Open File...

Congrats! You've made and opened your first HTML webpage.
