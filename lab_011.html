Hack4Impact Starter Pack

Lab 11 Making Changes

Goals

  • Learn how to monitor the state of the working directory

Change the README.md

Put the follow in the README.md:

README.md

#<Your Name>'s Personal Website
This is a personal website built using the Hack4Impact Starter Pack!
<You can add any description you want here.>

Note: .md files use markdown. It allows you to put formatting like italics or bold with just text. You might've seen markdown in Discord!

Check the Status

Now check the status of the working directory.

Execute

git status

Output

git status
(out)On branch main
(out)Changes not staged for commit:
(out)  (use "git add <file>..." to update what will be committed)
(out)  (use "git restore <file>..." to discard changes in working directory)
(out)	modified:   README.md
(out)
(out)no changes added to commit (use "git add" and/or "git commit -a")
