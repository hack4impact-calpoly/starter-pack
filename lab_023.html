Tables

In the early days of the Internet, tables were used to layout the pages. Now that we can use CSS, tables are mainly for displaying tabular data. Imagine an Excel or Google Sheets table and the kind of information you would display there. Tables have a table definition, table row, and table data cells. Respectively, these are <table>, <tr>, and <td>.
