  • Learn how to check the status of a Git repository

Check the Status of the Repository

Use the git status command to check and monitor the status of the repository in the current working directory.

Execute

git status

Output

git init
(out)On branch main
(out)
(out)No commits yet
(out)
(out)Untracked files:
(out) (use "git add <file>..." to include in what will be committed)
(out)        README.md
(out)
(out)nothing added to commit but untracked files present (use "git add" to track)

This message tells us that we have files in our working directory that have not been tracked. Specifically index.html. This means that the Git repository does not have any saved version of this file. An untracked file likely means a new file that has never been saved to the repository.
