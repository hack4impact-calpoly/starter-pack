Lab 4 Setup Visual Studio Code

Goals

  • Download and setup Visual Studio Code

Integrated Development Environment

Software developers use many different tools to write code, test programs, and manage large projects. Instead of installing each tool individually, we can use an integrated development environment or IDE. IDEs provide a convenient interface to install and manage all the tools we need to develop software.

One of the most popular IDEs is VSCode or Visual Studio Code by Microsoft. It's completely free, works on Windows and macOS, and has a strong open-source community providing features through downloadable extensions.

This is what Hack4Impact recommends new developers use and what you will need for the Starter Pack.

Install Visual Studio Code

Download and install Visual Studio Code from here: https://code.visualstudio.com/Download.

Install the Prettier Extension

Just like when writing an essay, you want your code to be readable and look nice. You can do this manually or you can use Prettier. Prettier is standard for formatting a variety of web development languages. For the purposes of the Starter Pack, we will need it to format HTML and CSS.

  1. Hover over the icons on the left-hand side. Click the Extensions icon.
  2. In the search bar, search for "Prettier".
  3. Click on the Prettier - Code formatter extension to see its details.
  4. Click the blue Install button to install the Prettier extension.

Setup Prettier

There's a little bit of setup to make sure that Prettier works as our formatter.

  1. Click the Settings gear in the lower-left of the screen
  2. Search for "Formatter" in the search bar
  3. Under the select menu for Default Formatter, choose Prettier - Code formatter
  4. Check the box that says Format on Save. This will apply formatting whenever you save a file.

Setup Autosave

If you've ever closed a Word assignment without saving, then you know the pain of not having autosave. Fortunately for us, VSCode comes with autosave. Unfortunately, it's disabled by default. Let's turn it on.

  1. Click the Settings gear in the lower-left of the screen
  2. Search for "Auto Save" in the search bar
  3. Under the select menu for Auto Save, choose afterDelay. This will automatically save your file after the delay time under Auto Save Delay. The default is 1000ms.
