Lab 31 Pseudo Classes and Pseudo Elements

Goals

  • Learn about pseudo classes and pseudo elements in CSS

Pseudo Classes

Pseudo classes are an extra add-on to CSS selectors that make selections that much more specific. They target an element's state or interaction. The ones we will talk about are link, visited, active, hover, and focus.



      
Pseudo Elements

      
Pseudo elements are attached in the exact same way as pseudo classes,
      but instead of targeting an element's state, it targets some part of the element's content
      that doesn't necessarily exist in the HTML structure. We'll cover first-letter,
      first-line, before, and 
      after.
