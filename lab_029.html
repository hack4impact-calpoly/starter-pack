CSS

A Cascading Style Sheet, or CSS, is the glitter, the paint, and the neon that makes a website shine. It provides all of the styling and visual appearance of a website. You might see some websites and developers using tools which mix HTML and CSS, however we strongly recommend that you keep them separate.

Remember, HTML is for content and CSS is for appearance. They both have their jobs and its your job as a developer to use them properly! With that said, let's learn about how to apply CSS to HTML.

Applying CSS

There are three ways to apply CSS to HTML: inline, internal, and external. Keeping with our separation of responsibilities, we will be using the external method of applying CSS

In the <head> of your index.html, add the <link> tag below.
