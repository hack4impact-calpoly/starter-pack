The Box Model

The box model represents the box that wraps around all HTML elements. It consists of the element, the padding, the border, and the margin. These all have properties that can be set in CSS.

The margin is the space outside of the box. The padding is the space inside the box. The border marks the edge between them.

Note: You don't have to use any of them, but a method for debugging HTML and CSS during web development is to set a border to a bright color to visually understand where the boxes lay on the webpage.

The border-color changes the border color. The border-width property takes a pixel value and controls the width of the border.
        The border-width property can also be set individually with border-top-width, border-right-width, border-bottom-width, and border-left-width.
