Make a GitHub account

GitHub is a website that allows developers to collaborate and work together on code using Git. Hack4Impact uses GitHub to hold all of our non-profit software projects. GitHub Discussions is a discussion board where you can ask questions and get help, interact with other prospective Hack4Impact developers, and meet your board!

GitHub will continue to be useful throughout your academic and professional career. Let's set up your account now!

  1. Go to the Github homepage: https://github.com/
  2. Click on the Sign up button in the top right corner.
  3. Enter your information, including your username, email address, and password.
  4. Follow the instructions to making your account!

Join Starter Pack Discussions

Access the Hack4Impact Starter Pack GitHub Discussions here: https://github.com/hack4impact-calpoly/starter-pack/discussions or on the home page.

Once on the page, make yourself known! Make an introduction in the pinned Introductions thread!

Note: If you ever need help, make sure to look through the Q&A discussions for others with similar issues or questions. If there are no answers, create a discussion and ask for help!
