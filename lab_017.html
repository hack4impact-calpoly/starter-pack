    Lab 17 Page Titles

    Goals

    • Add a <head> to your personal website
    • Add a <title> to your personal website
    • Update all your other pages to include the tags we've learned about

    Head

    Let's add some more structure to your website. While the main content of the your page goes inside it's <body>, there is also the <head> tag which goes before the <body>. The <head> contains information about the page which is not visible on the page itself.

    This includes the <title> tag which determines the title of your website, obvious we know. It can also include meta information for search engines to categorize your site with and tags that link your HTML to your CSS styling. For now, add the <head> tag before the <body>.

    Note: This is above your <body> tag, but inside your <html> tag.

    Website Title

    If you look at your website in your browser, you might notice that your tab just says index.html or the file path. That doesn't look very nice so let's change that.

    In your <head>, add a <title>. Then, between the opening and closing tags of the <title>, give your website a title. It can be anything!

    Now when you refresh your webpage, you should see that your website's tab has a title. If you save your webpage as a bookmark, this will also be the name of your bookmark! The page below has the title "Personal Website".

    Update Pages

    Let's set up your other HTML documents, just like you did for index.html. For each of the other HTML documents, add the following:

    • A document type declaration to the top of each file
    • Add an <html> tag to each page
    • Add a <head> tag to each page
    • Add a <title> in the <head> of each page
    • Add text to each <title>
    • Add a <body> to each page

    Note: If you need help, refer to the previous labs or ask on GitHub Discussions! Remember to stage and commit your changes often along the way!
