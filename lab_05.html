Lab 5 Setup Git

Goals

  • Download and setup Git to be ready for working with
  • Use the command line interface

Install Git

Git is a free and open source version control system or VCS. In the next lab, we will learn more about VCSs. For now, we will need to download Git and set it up for use. We will be using Git in a command line interface or CLI. Git also has several graphic interfaces, but learning to navigate a CLI is a useful skill for a software developer.

Windows users will use Powershell. Unix/Mac users will use Terminal.

Note: Git is a system and tool for managing code. GitHub is a proprietary platform and interface for hosting projects that use Git. Although used together, they are not owned together.

Check if you already have Git installed. Open your CLI (Powershell for Windows. Terminal for Unix/Mac.) and enter the following:

Execute:

git --version

Output:

git --version
(out)git version 2.42.0

If you don't see the expected output or similar, download and install Git from here: https://git-scm.com/downloads

Setup Name and Email

Now that we have Git installed, we need to configure it. In your CLI:

Execute

git config --global user.name "Your Name"
git config --global user.email "your_email@whatever.com"

Note: We recommend you use the same name and email as your GitHub account.

Setup Line Ending Preferences

For Unix/Mac users:

Execute:

git config --global core.autocrlf input
git config --global core.safecrlf true

And for Windows users:

Execute:

git config --global core.autocrlf true
git config --global core.safecrlf true
