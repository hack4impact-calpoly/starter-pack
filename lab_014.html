Lab 14 Committing Changes

Goals

  • Learn how to commit changes to the repository

Commit the Change

Run the following command to commit your changes from the staging area to your Git repository. Remember to include the -m flag.

Execute

git commit -m "feat: add content to README.md"

Output

git commit -m "feat: add content to README.md"
(out)[main c5ac52c] feat: add content to README.md
(out) 1 file changed, 4 insertions(+)
(out)

Check the Status

Let's check the status again to make sure it's committed.

Execute

git status

Output

git status
(out)On branch main
(out)nothing to commit, working tree clean
(out)

The working directory is clean and ready for you to continue!
