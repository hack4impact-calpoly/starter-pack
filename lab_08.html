Lab 8 Create a Project

Goals

  • Learn how to create a Git repository
  • Create the start of your personal website

Create a Working Directory

We will be using Visual Studio Code's integrated terminal to practice using Git, but everything you do in the VSCode terminal can also be done in your regular command line interface. VSCode has built-in graphical user interfaces (GUIs) for managing Git, as well as extensions that provide greater functionality, but for the purposes of better understanding Git we will be using the CLI.

  1. Open Visual Studio Code.
  2. Click the Open... text beneath Start or go to File > Open Folder... in the menu bar.
  3. Create a new folder called <user>.github.io with your GitHub username. This will be important for later~!
  4. Click Open to open the folder. This is your working directory.
  5. If you see a popup asking Do you trust the authors of the files in this folder?, click Yes, I trust the authors.

Create a File

This will be start of your personal website! We'll go over the details of HTML and CSS later, but for now let's create the main HTML file of your website.

  1. In the open folder, your Workspace, right-click the left-hand side Explorer panel to open the context menu and click New File... or click New File... on the Welcome page. You can also go to File > New File... in themenu bar.
  2. Name your new file README.md, create it, and save it to your open folder. If it's not open, click the file in the Explorer to open it.

This is a markdown file which will let us write formatted text.

Create the Repository

So far, all we've done is make a file in a folder on our system. We haven't really used any version control so let's change that. Open the VSCode integrated terminal with Ctrl + J on Windows or Cmd + J on macOS. You can also go to Terminal > New Terminal in the menu bar.

In the terminal, run the following to create a Git repository:

Execute

git init

Output

git init
(out)Initialized empty Git repository in Users/khoaly/khoa-l.github.io/.git/
